Lock the vote- An electronic voting system

Purpose

This web application has been created by Palak Jain and Vijay Mittal, as a part of course project for Web Database Developement-N510 in Spring 2022 at IUPUI.

Introduction

This project aims at developing an electronic-voting system specific to IUPUI that allows Students/Employees to securely cast votes and make their selection of candidate for a designated position. Our application stores the identity of all the voters and candidates using web application and the entire voting process can be easily managed by an authorized user. The main target of this project is to come with a leader of people choice through a fair judgement process.

Advantages of this system

Registration process is done by the voters through an application.
Easy OTP Verification.
Centralized database.
Provide security, privacy and ease of voting with reduced time and efforts.
Increase interest of students in all academic and cultural clubs.
Everyone in university will have equal right to vote and give their part to the system.

Resources used

For Front end: HTML, CSS
For Back end: PHP, MYSQL
Control end: JavaScript

Features

For Voters –
Self Registration
Login and Logout
Change Password
View all the Ongoing Active polls.
Cast vote for a selected candidate

For Admin –
Login and change password
Create new position for voting
Add new candidate for any active position
Enter details of voting (voting last date, voters, position etc.)
Modify the data
See Voting candidate report
